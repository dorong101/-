평형 반응식 르샤틀리에 원리
2NO₂ ⇌ N₂O₄ + 열

적갈색의 이산화질소(NO₂) 기체 2분자가 결합하여 무색의 사산화이질소(N₂O₄) 기체 1분자가 되며 열을 방출하는 가역 반응입니다.

실시간 입자 수
NO₂ (적갈색)
0
N₂O₄ (무색)
0
실험 미션 보드 0 / 3 완료
Mission 1. 피스톤 압축

피스톤을 아래로 밀어 부피를 줄여보세요. 르샤틀리에 원리에 의해 기체 분자 수가 감소하는 정반응이 일어나며 무색(N₂O₄) 입자가 증가합니다.

Mission 2. 흡열 반응 유도

실린더 온도를 450K 이상으로 올려 가열해보세요. 온도가 높아지면 열을 흡수하는 방향(역반응)으로 평형이 이동해 적갈색(NO₂) 입자가 늘어납니다.

Mission 3. 기체 주입과 평형

NO₂ 기체를 추가 주입하여 순간적으로 반응 지수 Q를 떨어뜨린 뒤, 새로운 화학 평형(Q = K)에 도달할 때까지 가만히 관찰해보세요.
